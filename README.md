CysterHood is a comprehensive web application designed to support PCOS awareness, preliminary triage, and personalized health insights through technology-driven analysis.

## Overview
The platform focuses on early-stage PCOS screening and education by combining structured assessments, AI-powered analysis, and accessible health information.

## Features

- **PCOS Preliminary Triage System**  
  Multi-step assessment tool to evaluate symptoms and risk factors.

- **Personalized Recommendation System**  
  AI-powered PCOS subtype classification with tailored health recommendations.

- **Blood Report Analysis Tool**  
  OCR-based extraction and analysis of hormone levels from uploaded medical report images.

- **Educational Content**  
  Curated, easy-to-understand information to improve PCOS awareness and understanding.

## Tech Stack

- **Backend:** Flask (Python)  
- **Frontend:** HTML, CSS (Glassmorphic UI), JavaScript  
- **ML / AI:** EasyOCR for text extraction from medical reports  
- **Styling:** Pink-themed glassmorphism with a premium aesthetic

  ## Features in Detail

### 1. PCOS Preliminary Triage System
Multi-step form that collects:
- Menstrual cycle information  
- BMI and physical metrics  
- Thyroid history  
- Family history  
- Blood reports analysis  

### 2. Personalized Recommendation System
- **Input:** Age, weight, height, BMI, pulse rate, cycle information  
- **Output:** PCOS subtype classification and personalized recommendations  

### 3. Blood Report Analysis
- Upload blood report images  
- OCR extraction of hormone levels  
- Automatic flagging of abnormal values  
- LH, FSH, AMH, RBS, BMI, TSH, HbA1c analysis
