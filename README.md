CysterHood - PCOS Support Network
A comprehensive web application for PCOS awareness, preliminary triage, and personalized health insights.

Features
PCOS Preliminary Triage System: Multi-step assessment tool
Personalized Recommendation System: AI-powered PCOS subtype classification and recommendations
Blood Report Analysis Tool: OCR-powered analysis of hormone levels from uploaded images
Educational Content: Comprehensive PCOS information
Tech Stack
Backend: Flask (Python)
Frontend: HTML, CSS (Glassmorphic design), JavaScript
ML/AI: EasyOCR for text extraction from medical reports
Styling: Pink-themed glassmorphism with premium aesthetics
